# CloverBloomMarket-WebProject
## Authors
* **Luka Tonia** (GitHub)

## Description
This app represents a final project in front-end development 2025 at Caucasus International University. The project was brought to its "finalized" form in 12 days.

## Features
* **Supermarket Simulation:** App mimics a functional Supermarket Page.
* **Responsive Design:** Allows comfortable usage on both desktop and mobile devices.
* **Functionality:** Key features of sorting, filtering, and displaying products are implemented.
* **Continuous Improvement:** One can always find room for improvement!

## Technologies Used
* HTML
* CSS
* JavaScript
