# CloverBloomMarket-WebProject
Authors
Luka Tonia  (GitHub)
Description
This app represents a final project in front-end  development 2025 at Caucasus International Unviersity. The project was brought to its "finalized" form in 12 days.

Features
App mimics the Supermarket Page
Responsive design let's comfortably use it both on desktop and mobile devices
Key features of sorting, filtering, and displaying products are implemented
One can always find room for improvement
Installation
Clone the repository: git clone https://github.com/LukaTonia/CloverBloomMarket-WebProject.git
Navigate to the project directory: cd CloverBloomMarket-WebProject
Install the dependencies: npm install
Usage
Start the development server: npm start
Open your browser and visit: http://localhost:3000
